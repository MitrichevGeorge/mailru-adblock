# mailru-adblock
ads blocker for mail.ru

## How to install
1. Download **Tampermonkey** from [![Firefox addons](https://img.shields.io/badge/Firefox_addons-8A2BE2?style=flat-square)](https://addons.mozilla.org/ru/firefox/addon/tampermonkey/)
2. Press **"Create a new script...** in tampermonkey
3. Paste the code from `script.js`
4. Save
